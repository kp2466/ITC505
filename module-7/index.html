HTTP Status Codes

  • 200 OK: The request has been succeeded.
  • 201 Created: The request is fulfilled and it got resulted in a new resource being created.
  • 400 Bad Request: The server cannot understand the request due to an invalid syntax.
  • 401 Unauthorized: The client need to authenticate himself to get the requested response.
  • 403 Forbidden: The client dont have any access rights to the content.
